NAME

mydebian - manual page for mydebian 0.1  

SYNOPSIS

mydebian ,COMMAND /[,SUBCOMMAND_OPTIONS/]  

DESCRIPTION

`mydebian` -- Post install for Debian.  

OPTIONS

-v, --version
Display version.
-h, --help
Display help.
 

COMMANDS

debian13|13|trixie [OPTIONS]
Post installation for Debian 13 (trixie).
debian14|14|trixie [OPTIONS]
Post installation for Debian 14 (forky).
add <gcc|python> <version> <priority>
Add alternate for gcc or python.
dowload_python <version>
Download python from python.org.
 

SUBCOMMAND OPTIONS

--list
List packages.
 

EXAMPLES

mydebian 13
Install package for debian 13.
mydebian 13 --list
Install and list package for debian 13.
mydebian add gcc 15 100
Add gcc alternative.
mydebian.sh add python 14 100
Add Python alternative.
